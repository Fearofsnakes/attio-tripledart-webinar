# outputs/

Where the skill runs land. Empty until the skills run. After a full pass this holds:

```
outputs/
  icp.md           # from icp-definition
  positioning.md   # from positioning-audit (scorecard + debt + white space)
  messaging.md     # from messaging-hierarchy (5-layer stack + 60-sec pitch)
  deck/            # from the playground skill (shareable visual)
```

## Pre-session (recommended)

Run all three skills once before the webinar and keep the outputs here. Two reasons:

1. If the live run stalls, you fall back to the saved artifacts and keep narrating without dead air.
2. You can sanity-check that the outputs are strong and on-voice before you present them.

Keep the live run anyway (it's the proof it's real), but have these as the safety net.
